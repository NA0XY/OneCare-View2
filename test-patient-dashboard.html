Testing Instructions

  1. Authentication Test: Start by logging in through the main login page (index.html) to simulate the full user flow.
  2. Navigation Test: Click on each page link above to verify all pages load correctly and navigation between pages works smoothly.
  3. Profile Dropdown Test: On each page, click the user profile dropdown to verify it opens/closes correctly and shows consistent styling.
  4. Profile Modal Test: Click "My Profile" in any dropdown to test the profile modal functionality and form submission.
  5. Settings Modal Test: Click "Settings" in any dropdown to test the settings modal with notification toggles and preferences.
  6. Mobile Responsiveness Test: Resize your browser or test on mobile devices to verify responsive design works across all pages.
  7. Feature Integration Test: Test specific features like medication management, health record viewing, and emergency contact functions.
  8. Logout Test: Use the logout function from any page to ensure it properly clears authentication and redirects to login.
